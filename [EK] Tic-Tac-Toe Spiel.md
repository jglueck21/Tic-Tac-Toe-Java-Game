# [EK] Tic-Tac-Toe Spiel

- Regelmäßige Commits

- Sinnvolle Commitmessages





Gegeben ist ein 3x3 Feld mit Buttons. Die Spieler können gegenseitig auf eines der Felder ihr Symbol setzten. Ziel des Spieles ist entweder Vertikal, Horizontal oder Diagonal 3 mal ihr Symbol in einen Reihe zu haben um das Spiel zu gewinnen.
